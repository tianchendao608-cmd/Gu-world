Gu-world V8.5《诡域历练与斗蛊战场》安装说明

最稳安装：
1. 先在 GitHub 备份你现在的 app.js。
2. 把本包里的 app.js 上传并覆盖仓库根目录的 app.js。
3. 打开 style.css，把 gu_world_v85_style.css 里的全部内容复制到 style.css 最底部。
4. Commit changes。等 GitHub Pages 刷新后，打开网站强制刷新。

这版更新内容：
- 历练谷全部换成五大诡域副本：血狼林、枯骨坟场、黑风峡谷、幽魂沼泽、万蛊深渊。
- 每张地图都有大地图、障碍、妖兽图案、妖兽形状、技能、掉落。
- WASD/方向键移动，手机端方向盘移动。速度属性会影响移动速度。
- 妖兽会追击、近战、释放技能。鬼面蛛会定身。
- 斗蛊台加入战场地图、双方位置、距离限制。
- 手机底部战斗栏默认收纳，点击“战斗栏”展开装备快捷栏。

回滚方法：
如果网站出错，把备份的旧 app.js 覆盖回来，并删除 style.css 底部 V8.5 样式即可。
