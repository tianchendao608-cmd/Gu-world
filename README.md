# 五域两天 · V7.5 勾选多选与蛊方材料更新

替换 GitHub 仓库根目录里的：

- app.js
- style.css

更新内容：

1. 创建杀招、凡蛊屋时，“所需蛊虫”从原来的 Shift/Ctrl 多选，改为卡片勾选式多选。
2. 每个蛊虫后面都有小框，勾上就是选中，手机和电脑都更方便。
3. 蛊方创作新增“所需蛊虫”和“所需蛊材”勾选区。
4. 管理员创建蛊方时可选择全库蛊虫、全库蛊材。
5. 普通玩家创建蛊方时只能选择自己背包里已有的蛊虫、蛊材。
6. 蛊方详情页会显示所需蛊虫、所需蛊材。

替换后等 GitHub Actions 部署完成，再在网页按 Command + Shift + R 强制刷新。
