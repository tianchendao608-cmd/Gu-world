修复/新增：

1. 详情页显示原始属性，不再把跨流派属性直接除半显示。
2. 跨流派使用仍会减半，但 +1 不会变成 0。

4. 同一蛊虫/杀招/凡蛊屋不能重复装备到多个格子。
5. 未吸收蛊物不能装备；已吸收且装备后，乾坤袋显示“被装备”。
6. 创建杀招：任意主/副流派达到大师即可。
7. 创建凡蛊屋：任意主/副流派达到宗师即可。
8. 普通玩家三转以上才可创建蛊虫、蛊方、势力。
9. 创建杀招/凡蛊屋的蛊虫选择改成卡片式多选，更适合手机。
