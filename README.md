# 五域两天 · V7.2 装备栏与交互优化

替换 GitHub 仓库里的：

- app.js
- style.css

更新内容：

1. 蛊虫详情显示距离、冷却时间、持续/定身时间。
2. 编辑蛊虫、杀招、凡蛊屋时可填写距离、冷却、持续。
3. 元石恢复真元改为一键恢复，默认每次1颗元石；管理员可设置一次使用数量。
4. 右下角新增 0-9 装备栏，蛊虫、杀招、凡蛊屋可装备。
5. 装备栏显示名称和最高属性，点击使用后扣真元并进入冷却倒计时。
6. 交易坊可摆上元石与背包里的蛊虫、杀招、凡蛊屋、蛊方、蛊材。
7. 势力殿可创建势力，普通玩家三转以上，管理员不限。
8. 传音阁新增公共聊天、群聊、私聊入口。

注意：这版仍是前端简易交易台，复杂的双方背包自动结算以后可以继续强化。
