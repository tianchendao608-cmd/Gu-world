# 五域两天 V7.4 一次性稳定整合版

替换 GitHub 根目录：

- app.js
- style.css

更新内容：

1. 修复跨流派属性显示被直接除二；只在使用时减半，并且 +1 不会变 0。
2. 详情页显示距离、冷却时间、持续/定身时间。
3. 一键恢复真元，默认 1 颗元石；管理员可设置一次恢复元石数量。
4. 0-9 十格装备栏，手机默认显示 5 格，可弹窗打开 10 格。
5. 未吸收蛊物不能装备，同一蛊物不能重复装备，已装备显示“被装备”。
6. 创建杀招/凡蛊屋的所需蛊虫选择变成简洁多选。
7. 任意流派大师可创杀招，任意流派宗师可创凡蛊屋。
8. 三转以上可创建蛊虫、蛊方、势力。
9. 交易坊支持元石与背包蛊物交换，双方确认后结算。
10. 传音阁支持公共聊天、群聊、私信与好友入口。

替换后等待 GitHub Pages 部署完成，再按 Command + Shift + R 强制刷新。
