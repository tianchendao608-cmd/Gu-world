# 五域两天 · 蛊虫资源包 v1

内容：
- 蛊虫：130 只
- 蛊材：104 种
- 蛊方：130 份
- 流派：血道, 云道, 火道, 雷道, 魂道, 天道, 运道, 炼道, 变化道, 风道, 沙道, 气道, 力道

规则：
- 一至五转
- 吸收所需真元 = 使用一次真元 × 5
- 属性范围符合：一转1-5，二转3-10，三转8-20，四转15-30，五转20-50

使用：
1. 把 `import_resource_pack.html` 上传到 GitHub 仓库根目录。
2. 打开：`你的GitHubPages网址/import_resource_pack.html`
3. 点击“一键导入 Firestore”。
4. 回主站刷新。

注意：重复导入会覆盖同ID资料，不会重复生成一堆乱数据。
