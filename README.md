# 五域两天 V8.3 历练与突破强化版

替换 GitHub 根目录：

- app.js
- style.css

更新内容：

1. 历练谷改为每个试炼地独立 1 分钟冷却。
2. 新增多个一转、二转试炼地。
3. 低转可强闯高转试炼地，但只扣血无收益。
4. 高转刷低转试炼地只获得 25%收益。
5. 闭关洞大境界突破新增“真元冲关 / 元池丹田”动画。
6. 突破时元池颜色会在 20 秒内逐渐从当前真元颜色转向下一转真元颜色。
7. 突破期间真元逐秒扣除，真元不足会停止并失败。
8. 突破期间仍可用元石恢复真元，真元和生命每秒刷新显示。
9. 保留 V8.2 的死亡、重生、自动恢复、技能冷却倒计时规则。
